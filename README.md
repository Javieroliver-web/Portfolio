# Portfolio Personal Interactivo

¡Bienvenidos al lanzamiento oficial de mi portfolio personal! Este proyecto representa mi visión como desarrollador Front-end, combinando diseño moderno con funcionalidad intuitiva. Desarrollado con las últimas tecnologías web, este portfolio no solo muestra mi experiencia y habilidades, sino que también demuestra mi capacidad para crear interfaces de usuario atractivas y responsivas.

## Características Principales

-   **Modo Claro/Oscuro**: Interfaz adaptable con cambio de tema.
-   **Personalización de Colores**: 4 esquemas de color diferentes para elegir.
-   **Multilingüe**: Soporte para español e inglés, cargados dinámicamente.
-   **Galería de Proyectos**: Carrusel interactivo con botones para mostrar múltiples imágenes por proyecto.
-   **Visor de Imágenes**: Previsualización en pantalla grande (lightbox) para las imágenes de los proyectos.
-   **Diseño Responsivo**: Adaptable a todos los dispositivos, desde móviles hasta escritorio.
-   **Sección de Experiencia**: Muestra experiencia laboral relevante.
-   **Habilidades Técnicas**: Visualización interactiva de competencias.
-   **Enlaces Directos**: Conexiones a redes sociales y CV descargable.

## Tecnologías Utilizadas

-   HTML5
-   CSS3
    -   Variables CSS
    -   Flexbox
    -   Grid
    -   Animaciones
-   JavaScript
    -   Manipulación del DOM
    -   Event Listeners
    -   Async/Await (para carga de idiomas)
    -   Fetch API

## Características Técnicas

-   **Internacionalización (i18n)**: Carga dinámica de contenido en español e inglés desde archivos JSON.
-   **Interactividad**: Cambio dinámico de temas y colores.
-   **Galería Interactiva**: Carrusel de imágenes con controles de JavaScript y `scroll-behavior: smooth`.
-   **Visor Lightbox**: Modal de JS para previsualización de imágenes, que bloquea el scroll del body.
-   **Animaciones**: Barras de progreso animadas para habilidades.
-   **Optimización**: Código limpio y bien estructurado.
-   **Accesibilidad**: Estructura semántica HTML.
